If a Tree Falls in the Forest

If A Tree Falls In The Forest is a nature and horror themed game where you wander around a forest encountering the strange phantoms of deceased wildlife. It was a short game built for the show "Abandoned Places, Forgotten Spaces" in just one month. It was built by KJ Martinet and Salomao Becker.
